# Pyrus v0.5.0 — Early Access

Pyrus is now available as a free Early Access release.

Pyrus is a calm, local-first reading space for Markdown knowledge bases. It helps you return to what matters, continue where you stopped, and find any note without sending your knowledge anywhere.

## Highlights

- Create a new knowledge base with a starter `Welcome.md`, or open an existing Markdown folder.
- Continue reading from the exact place where you stopped.
- Pin important notes on the knowledge home.
- Search file names and document content with `⌘K` on macOS or `Ctrl+K` on Windows/Linux.
- Jump directly to matched content with a short visual highlight.
- Follow long documents with a live outline and reading-progress indicator.
- Keep everything local: no account and no cloud sync required.

## Early Access

This release is intended for early readers. If something feels unclear, breaks, or could make reading better, please use **Send feedback** in Preferences or open an issue on GitHub.
